# Zapchasty — статический сайт

Простой статический сайт на HTML, CSS и JavaScript.

## Структура

```
zapchasty/
├── index.html      # Главная страница
├── css/
│   └── style.css   # Стили
├── js/
│   └── main.js     # Скрипты
├── images/         # Изображения (пустая папка)
└── README.md
```

## Запуск

Откройте `index.html` в браузере или используйте любой локальный сервер, например:

- **Python:** `python -m http.server 8000` (в корне проекта)
- **Node (npx):** `npx serve .`

Затем откройте в браузере: `http://localhost:8000`

## Настройка

- Тексты и блоки редактируются в `index.html`
- Оформление — в `css/style.css` (переменные в `:root`)
- Картинки складывайте в папку `images/` и подключайте в разметке
